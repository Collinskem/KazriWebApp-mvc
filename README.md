#Email Notifications
whenever a staff profile is created, edited, or deleted; they should be able to receive a personalized email notification as below.

Profile Created: 
Mail Subject: Profile Notification #Created
Mail Body: Greeting <Insert Name>, we are glad to inform you that your staff profile has been created.

Profile Edited: 
Mail Subject: Profile Notification #Edited
Mail Body: Greeting <Insert Name>, we are glad to inform you that your staff profile has been edited.

Profile Deleted: 
Mail Subject: Profile Notification #Deleted
Mail Body: Greeting <Insert Name>, we are sad to inform you that your staff profile has been deleted
